# Vizsgaremek
Böjtös Attila, Kuik Filep Balázs, Horváth Boti

Vizsgánk témája egy Steam-hez hasonló játékajánló webalkalmazás, ami a felhasználó ízlésvilágát ia követi.
Tartalmazni fog egy regisztrációs/belépő oldalt, azt követi maga a webalkalmazás. Ahol eleinte különböző játékokat dob fel, értékeléstől függetlenül. Keresni is lehet majd a felületen, pl.: játék névre (értékelés szerint is), és játék kategóriára is.
A felhasználó adatait eltárolja egy adatbázisban, valamint azokat a játékokat is, amikre rákeresett, vagy csak megjelölte kedvencként. Ezek alapján fogja a későbbiekben ajánlani a felhasználó ízléséhez illő játékokat.
